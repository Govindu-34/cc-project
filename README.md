# ☁️ AKTech — Cloud Computing Platform

AKTech is a modern cloud computing platform UI inspired by real-world services like AWS, Azure, and Google Cloud. This project focuses on creating a high-quality frontend experience with advanced design, animations, and structured cloud service representation.

---

## 🚀 Features

- 🌐 Modern cloud platform UI design  
- ✨ Glassmorphism and smooth animations  
- ⚡ Interactive 3D and dynamic elements  
- 📊 Live dashboard simulation (CPU, memory, network stats)  
- 🧩 Modular sections (Services, Architecture, Pricing, Dashboard, About)  
- 📱 Fully responsive design  

---

## 🧱 Sections Included

### 🏠 Home
- Hero section with cloud concept
- Real-time stats (uptime, servers, regions)
- Trusted companies showcase

### 🛠 Services
- Compute, storage, networking, AI services
- Feature cards with descriptions

### 🏗 Architecture
- Layered cloud system design
- Load balancing, Kubernetes, compute layers

### 💰 Pricing
- Starter, Pro, and Enterprise plans
- Clear and simple pricing structure

### 📊 Dashboard
- Simulated analytics (CPU, memory, traffic)
- Region usage and logs

### 👤 About
- Project details and team structure

---

## 🛠 Tech Stack

- **HTML5** — Structure  
- **CSS3** — Styling (Glassmorphism, animations)  
- **JavaScript** — Interactivity  
- **Three.js** — 3D animations and visual effects

## 🎯 Purpose of the Project

This project is built to:

- Demonstrate frontend development skills  
- Showcase modern UI/UX design  
- Simulate a real-world SaaS/cloud platform  
- Understand how cloud platforms are structured
