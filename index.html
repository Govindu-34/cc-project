Project Overview

AKTech is a cloud computing platform prototype developed as a comprehensive academic project. It models a real-world Infrastructure-as-a-Service (IaaS) and Platform-as-a-Service (PaaS) provider, covering the full stack from physical hardware abstraction to developer-facing APIs.

The project covers key cloud computing concepts including virtualization, distributed storage, container orchestration, networking, security, observability, and cost management — all presented through a modern, production-quality web interface.

Cloud Computing Distributed Systems Kubernetes Networking Security IaaS / PaaS
Project Timeline
PHASE 1 — Research
Literature review of cloud architectures, IaaS/PaaS models, and existing platforms (AWS, GCP, Azure).
PHASE 2 — Design
Architecture design, service catalog definition, pricing model, and system component specification.
PHASE 3 — Development
Full-stack implementation of the platform UI, live dashboard, and interactive service explorer.
PHASE 4 — Submission
Final documentation, presentation, and project submission with live demo.
